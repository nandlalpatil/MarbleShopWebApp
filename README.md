# Marble Shop Web Application

Marble Shop Web Application is a web-based system developed using **ASP.NET MVC (.NET Framework)** and **ADO.NET** to manage marble products, customer orders, and invoices efficiently. The application allows administrators to add and manage products, while customers can place orders through a simple cart interface.

## Features

* Product Management (Add, Edit, Delete, Product List)
* Product Image Upload
* Cart System for selecting products
* Order Management
* Customer Details Capture (Name, Mobile Number)
* QR Code Scanner for quick product selection
* Invoice Generation
* PDF Invoice Download using Rotativa
* Search functionality for orders
* Session-based login authentication

## Technologies Used

* ASP.NET MVC (.NET Framework)
* C#
* ADO.NET
* SQL Server
* JavaScript & jQuery
* Bootstrap
* Rotativa (for PDF generation)

## Modules

1. Product Management
2. Cart System
3. Order Management
4. Invoice & PDF Generation
5. User Management

## Purpose

The purpose of this project is to simplify the process of managing marble shop inventory and customer orders while providing a clean and user-friendly interface.

---

Developed as a learning and practical project to demonstrate **ASP.NET MVC development, database operations with ADO.NET, and web application design**.
